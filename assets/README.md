# Assets Directory

This directory contains static assets used across the Prompt Diary platform.

## Structure

```
assets/
├── images/       # Platform images, logos, icons
├── videos/       # Demo videos, tutorials
└── README.md     # This file
```

## Usage

- Frontend assets should be placed in `frontend/public/` for Next.js serving
- Shared design assets (logos, brand guidelines) go here
- Marketing materials and screenshots go here
